# AI Homework - Face Recognition

## Mô tả
Bài tập nhận diện khuôn mặt sử dụng Python và mô hình AI/CNN trên Google Colab.

## File chính
- `AI_HOMEWORK_FACE_RECOGNITION.ipynb`: file Colab chứa toàn bộ code huấn luyện và nhận diện khuôn mặt.

## Cách chạy
1. Mở file `.ipynb` bằng Google Colab.
2. Chạy lần lượt các cell từ trên xuống.
3. Upload ảnh/dataset theo hướng dẫn trong notebook.
4. Chương trình sẽ train model và hiển thị kết quả nhận diện.

## Kết quả
Mô hình có thể nhận diện khuôn mặt dựa trên dữ liệu đã train.
