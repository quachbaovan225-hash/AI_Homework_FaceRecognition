# Face Recognition using CNN

## Mô tả
Đây là bài tập nhận diện khuôn mặt sử dụng mô hình CNN trên Google Colab.

## Chức năng
- Load dataset khuôn mặt
- Tiền xử lý ảnh
- Train mô hình CNN
- Đánh giá độ chính xác accuracy
- Test nhận diện khuôn mặt bằng ảnh mới

## Dataset
Dataset gồm các ảnh khuôn mặt được chia theo từng người/lớp.

Do dataset đầy đủ có dung lượng lớn nên em để dataset tại Google Drive:

Link dataset: https://drive.google.com/drive/folders/1_X7A5IrsR2qyAnyyM-gTS0c5nelWN-q5


## Cách chạy
1. Mở file `AI_HOMEWORK_FACE_RECOGNITION.ipynb` bằng Google Colab.
2. Chạy lần lượt các cell từ trên xuống.
3. Nếu dùng dataset từ Google Drive, cập nhật lại đường dẫn dataset trong code.
4. Sau khi train, chương trình sẽ in ra accuracy.
5. Upload ảnh mới để kiểm tra kết quả nhận diện.
